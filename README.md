# 🧠 Actividad 4: Manejo de Datos con Pandas

Este proyecto demuestra el uso práctico de **Pandas** y **NumPy** en Python para la manipulación, limpieza y análisis de datos tabulares.  
A lo largo del código se realizan operaciones fundamentales como la creación de **Series**, **DataFrames**, manejo de **valores faltantes**, **filtrado**, **ordenamiento**, **estadísticas básicas** y **exportación de datos** a archivos CSV.
